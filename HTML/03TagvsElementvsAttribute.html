Tag vs Element vs attributes

Tag

Tag which is come in between < angularbrackets >

90% tags come with start tag and end tag i.e we have to close a tag. like h1, p, div, etc

some tag does not require any closing tag like br, hr, img etc. often called as void tag or Empty tag or self closing tag.

Element

Start tag - some content - End tag

attributes

It provides any information or used for modification of an element and always written in start tag.

Elements in HTML have attributes; these are additional values that configure the elements or adjust their behavior in various ways to meet the criteria the users want.
